## 📌 Project Overview

This project builds an end-to-end Machine Learning solution to predict **customer churn** in the travel industry using the **Random Forest** algorithm. It includes data preprocessing, model training, evaluation, visualization, and deployment via **Streamlit Cloud**.

## 🗃️ Dataset Description

**File:** `customer_churn.csv`  
**Rows:** 954 | **Columns:** 7

| Column | Type | Description |
|--------|------|-------------|
| Age | Numeric | Customer age |
| FrequentFlyer | Binary (Yes/No) | Is the customer a frequent flyer? |
| AnnualIncomeClass | Categorical | Low / Middle / High Income |
| ServicesOpted | Numeric | Number of additional services opted |
| AccountSyncedToSocialMedia | Binary (Yes/No) | Social media account linked? |
| BookedHotelOrNot | Binary (Yes/No) | Hotel booked via platform? |
| **Target** | Binary (0/1) | **0 = No Churn, 1 = Churn** |

---

## 🧠 ML Workflow

1. **Data Cleaning** — Fix encoding noise, convert data types
2. **Feature Encoding** — Map Yes/No to 1/0, ordinal encode income
3. **Train-Test Split** — 80/20 with stratification
4. **Random Forest** — 100 trees, balanced class weights
5. **Evaluation** — Accuracy, Confusion Matrix, ROC AUC, Classification Report
6. **Visualizations** — Churn distribution, feature distributions, correlation heatmap

## 🚀 How to Run Locally

### 1. Clone the repository
```bash
git clone https://github.com/YOUR_USERNAME/churn-prediction.git
cd churn-prediction
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Run the Jupyter Notebook first (generates model.pkl)
```bash
jupyter notebook Customer_Churn_Prediction.ipynb
```

### 4. Run the Streamlit App
```bash
streamlit run app.py
```

---

## ☁️ Streamlit Cloud Deployment

1. Push all files to a **GitHub repository**
2. Go to [streamlit.io/cloud](https://streamlit.io/cloud)
3. Click **"Deploy App"**
4. Select your repo and set `app.py` as the main file
5. Click **Deploy** — your app will be live at `https://your-app.streamlit.app`
